# 🌐 BİRDY — Neural Intelligence Platform

[![License: MIT](https://img.shields.io/badge/License-MIT-blue.svg)](https://opensource.org/licenses/MIT)
[![Version](https://img.shields.io/badge/Version-4.0-kinetic_cyan.svg)]()

BİRDY is a high-performance autonomous AI interface designed for deep OS integration and complex task execution. It bridges the gap between human intent and machine action using advanced neural processing. Inspired by JARVIS from Iron Man, BİRDY serves as a complete AI assistant with voice interaction, visual interface, and autonomous capabilities.

---

## 🚀 Key Features

- **Neural Hub (BİRDY):** Real-time voice and visual interaction with advanced reasoning.
- **Autonomous Planning:** Deconstructs complex objectives into efficient execution pipelines.
- **Deep System Sovereignty:** Full control over apps, files, browser, and system settings.
- **Proactive Intelligence:** Anticipates user needs and provides real-time system health monitoring.
- **Visual Tele-sync:** Analyzes and responds to visual data from screen and camera.
- **JARVIS-like Interface:** Cinematic visual interface with kinetic animations and HUD elements.
- **Voice Interaction:** Natural conversation with AI personality matching JARVIS characteristics.

## 🏗️ System Architecture

- **`central_nerve`**: The strategic reasoning engine (Planner) and execution hub.
- **`ability_core`**: A suite of autonomous modules for real-world system interaction.
- **`visual_link`**: Advanced neural interface for visual data processing.
- **`neural_store`**: Persistent long-term memory synapse using local vector stores.
- **`security_vault`**: Encrypted layer for sensitive API and credential management.
- **`system_laws`**: Core directives and personality protocols for JARVIS.

## ⚡ Ignition Protocol

1. **Prerequisites:** Ensure Python 3.13+ is installed and your mic/speaker are active.
2. **Setup:** Run `installer.bat` to synchronize all neural dependencies.
3. **Authorization:** Enter your Google Gemini API Key in the secure vault when prompted on first launch.
4. **Ignition:** Launch `run.bat` to ignite the BİRDY Neural Engine.
